CSS Review

Let's add a style sheet. After we add a style sheet we will modify properties of the stylesheet

Actions to take

  • link the stylesheet called stylesheet.css in between the head tags
    1. Add line <link rel="stylesheet" type="text/css" href="css/stylesheet.css" />

  • Reviewing using stylesheet, font properties, and selectors
    1. Select all elements in the body with body{} selector. Note - style cascades from body to all elements in the body
    2. Inside the curly braces add font-family: Arial, sans-serif;
    3. Select main_sentence element using the id selector and make the font size 20px: #main_sentence{ font-size: 18px;}
    4. Select underlined class elements using the class selector and add the text-decoration to it .underlined{ text-decoration: underline;}

  • Review alignment and padding
    1. Add tag h1{ text-align: center; }
    2. Add a border around the section element for more differentation using section{ border: 5px solid black; border-radius: 10px; }
    3. Add padding to section element to move the inside of the section away from the border with section{ padding: 30px;}

  • Background Style
    1. Add background-color to section to give it style using: section{ background-color: rgb(220,182,193) }
    2. Add background-image to the header using: background-image: url("images/banner-1763839_340.jpg")
